# Holberton School - Web Front-End

This repository contains all the projects completed during the **Web Front-End** curriculum at Holberton School. The focus is on mastering the fundamental technologies of the web‚Äî**HTML, CSS, and JavaScript**‚Äîwith an emphasis on modern web development practices, responsive design, and toolchains.

The projects progress from basic semantic HTML and simple styling to advanced CSS techniques, responsive layouts, CSS pre-processors (SASS), and modern asset bundling with Webpack.

## ‚öôÔ∏è General Requirements

All projects in this repository adhere to the following general standards:

  * **HTML:** All HTML files must pass **W3C validation**.
  * **CSS:** All CSS files must pass **W3C validation** and use correct naming conventions.
  * **JavaScript:** Scripts are written in modern JavaScript (ES6+).
  * **Best Practices:** Code must be semantic, follow best practices for accessibility, and be structured for maintainability.
  * **Cross-Browser:** Projects are tested for compatibility across modern web browsers.
